# TodoSchedule

## 1. Introduction
간단한 스케줄 관리 프로그램입니다.
kizitonwose의 calendar 라이브러리를 사용하여 달력을 구현하였습니다.
Room 라이브러리를 사용하여 데이터베이스를 구현하였습니다.
MVVM 아키텍처를 사용하여 구현하였습니다.
dagger hilt를 사용하여 의존성을 주입하였습니다.

## 2. Features
- 일정 추가
- 일정 수정
- 일정 삭제
- 일정 확인
